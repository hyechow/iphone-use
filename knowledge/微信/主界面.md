# 微信 (82)

微信主界面，展示聊天列表及底部导航栏，提供新建会话、搜索联系人及切换不同功能模块入口。

- [新建按钮] 右上角 → 弹出包含社交与支付功能的下拉菜单
- [搜索栏] 列表区域顶部 → 进入搜索页面以查找联系人或内容
- [设备管理入口] 列表区域顶部 → 进入设备管理页面查看登录状态并退出
- [聊天列表项] 列表区域 → 进入对应联系人、群组或公众号的聊天详情页
- [通讯录标签] 底部导航栏左侧 → 进入通讯录页面管理联系人分组
- [发现标签] 底部导航栏中间 → 进入发现页访问朋友圈等功能
- [微信标签] 底部导航栏左侧（当前选中） → 刷新聊天列表并更新未读消息状态
- [我标签] 底部导航栏右侧 → 进入个人中心查看资料及设置
