# 微信 (82)

微信主界面展示聊天列表及底部导航栏，提供新建会话、搜索功能及多入口切换。

- [新建按钮] 右上角 → 弹出包含发起群聊、添加朋友等功能的菜单
- [搜索栏] 列表区域顶部 → 进入支持文本或语音输入的搜索页面
- [设备管理入口] 列表区域顶部 → 进入设备管理页面查看状态并退出登录
- [聊天联系人条目] 列表区域 → 进入对应联系人的聊天详情页查看历史消息
- [群组聊天条目] 列表区域 → 进入对应群组的聊天详情页查看消息和通知
- [公众号/服务号条目] 列表区域 → 进入公众号或服务号的聊天详情页查看推送内容
- [通讯录标签] 底部导航栏 → 进入通讯录页面管理联系人分组
- [发现标签] 底部导航栏 → 进入发现页访问朋友圈、视频号等功能
- [微信标签] 底部导航栏 → 刷新并展示最新的聊天列表及未读消息
- [我标签] 底部导航栏 → 进入个人中心查看资料及访问服务收藏
